# Recursive Knowledge Cycle Theory (RKCT)

The Recursive Knowledge Cycle Theory (RKCT) is an open theoretical framework for describing transition processes in which knowledge systems generate new forms of organization, reproduction, and self-structuring. RKCT aims to identify recurring patterns across biological, cultural, and technological domains.

This repository contains the documentation, conceptual models, and bilingual materials (EN/DE) associated with the development of RKCT.

## Structure

- **/de/** – German version of the RKCT materials  
- **/en/** – English version of the RKCT materials  
- **/index.html** – Language selection page  
- **/de/rkct-4.4.html** – Core model (German)  
- **/en/rkct-4.4.html** – Core model (English)

## Purpose

The goal of this project is to provide a clear, accessible, and evolving reference for the RKCT, including:

- theoretical foundations  
- versioned model descriptions  
- terminology and glossary  
- background and motivation  
- links to external scientific resources (e.g., OSF)
